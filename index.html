QuickStart with Python

  1. Download HF Trading Simulator from the following page: download
  2. Unpack zip archive into local folder and navigate into it
  3. Setup python-3 for windows from https://www.python.org/downloads/
    * for debian/ubuntu sudo apt-get install python3
    * for centos/fedora sudo yum install python3

    4. There are binaries in the archive: tsim_get and tsim_play
  4. run tsim_get demo to download free demo day

    5. There are some examples in py_examples folder
  5. Run tsim_play -ohlc py_examples/sma_ohlc.py
  6. Report will be generated in JSON format in reports folder
  7. Run tsim_play py_examples/simple_hft.py for hft backtesting
  8. If you have real trade module and want to run it on real exchange, then:
    1. copy conf/tsim_example.toml into conf/tsim.toml and then edit
    2. tsim_play -run -ohlc py_examples/simple_sma.py
  9. For creating your own strategy, please move the next slide slide
